# Unity 2D 足球游戏

基于 Unity 开发的多人在线足球游戏。玩家可选择不同国家队进行比赛，玩法参考 EA FC / FIFA Ultimate Team（UT）模式。

## 游戏操作

默认键位如下（可在游戏内修改）：

| 按键          | 功能                     |
| ------------- | ------------------------ |
| W / A / S / D | 移动                     |
| Q             | 直塞                     |
| E             | 短传                     |
| 4             | 长传，铲球               |
| X             | 切换球员                 |
| 空格          | 射门，大力射门，凌空射门 |

## 游戏模式

创建房间时可选择以下模式：

- 正式比赛
- 自由训练
- 对抗训练

## 字体资源

项目使用 TextMeshPro 字体系统。通过 **TextMeshPro Font Asset Creator** 生成对应字体资源。
